# PDF-Based Question Answering Chatbot 

A simple AI chatbot built with Flask, LangChain, and OpenRouter API, capable of answering questions.

## Features

- Chat interface to ask any question  
- AI-powered responses using LangChain and OpenRouter  
- Clean and minimal Flask backend  
- Ready demo use  

## Project Structure
smart_assistant

  -app.py
  
  -templates
  
    -index.html
    
  -requirements.txt


## Installation Instructions

1. Create a virtual environment
python -m venv venv

2. Activate the virtual environment
venv\Scripts\Activate.ps1

3. Install dependencies
pip install -r requirements.txt

4. Running the Project
python app.py
